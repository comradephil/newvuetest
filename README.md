# newvue

## Client 

### Setup
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```
### Compiles and minifies for production
```
npm run build
```
### Lints and fixes files
```
npm run lint
```
## Server 

### Setup
```
npm install
```
### Compiles for development
```
node server.js
```
### Forever/nodaemon for prod
```
forever server.js
